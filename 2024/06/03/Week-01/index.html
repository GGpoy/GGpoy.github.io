基础知识

  1. 二进制

    计算机中，每一个0或1，叫做一个比特（bit）。

    十进制转二进制:除2取余法

    二进制转十进制：按权相加法

  2. 计算机的存储单元

    最小的信息存储单元：位（比特位、bit），小写字母b表示

    最小的数据存储单元：：字节（byte），大写字母B表示

    1B=8b
    1KB = 1024B
    1MB = 1024KB
    1GB = 1024MB
    1TB = 1024GB

  3. 标识符

    给包、类、方法 、变量等起的名字

    组成规则：
    字母、数字、下划线、$符号组成
    不能以数字开头
    不能为JAVA中关键字

    命名规则：
    包：域名倒过来，所有字母小写
    类：单词首字母大写（驼峰命名）
    方法：首单词的首字母小写，其他单词首字母大写
    常量：所有字母大写

  4. 数据类型

    基本数据类型：int、short、long、float、double、char、byte、boolean

    引用数据类型：类、对象、数组等

    基本数据类型比较“值”，引用数据类型比较“地址”

    数据类型** 关键字 内存占用 取值范围 备注
    字节型 byte 1个字节 -128~127 8位，2^7-1
    短整型 short 2个字节 -32768~32767 16位，2^15-1
    整型 int（整数默认） 4个字节 -2的31次方~2的31次方-1 默认值为0
    长整型 long 8个字节 -2的63次方~2的63次方-1 定义long类型的数据时，要在数据后面加L
    单精度浮点数 float 4个字节 1.4013E-45~3.4028E+38 定义float类型的数据时，要在数据后面加F
    双精度浮点数 double（浮点数默认） 8个字节 4.9E-324~1.7977E+308 默认值为0.0
    字符型 char 2个字节 0-65535 用单引号，例如’A’,’b’,’陈’
    布尔类型 boolean 1个字节 true，false 默认值为false

    注意：浮点型可能只是一个近似值，并非精确的值。如0.09+0.01=0.9999999….

  5. 数据类型转换

    取值范围小的数据类型与取值范围大的数据类型进行运算,会先将小的数据类型提升为大的,再运算

    强制类型转换的格式：
    b = (byte)(a + b);
    注意：如果超出了被赋值的数据类型的取值范围得到的结果会与你期望的结果不同

  13. static

    成员变量按照有无static修饰分为类变量和实例变量

    类变量的访问：类名.类变量（推荐）、对象名.类变量（不推荐）

    类变量与类一起加载一次，在内存中只有一份，被类的所有对象共享，生命周期与类共存亡

    类方法的好处：

    实例方法的调用需要创建对象去调用，而对象会占用堆内存，若此时该对象仅仅为了调用该方法，就会浪费内存。
    而类方法直接使用类名调用，jvm会直接在方法区执行方法，不占用堆或栈的内存。
    tips：在创建工具类时，例如上面的验证码的生成，建议使用私有构造函数。仅通过类名去调用工具类方法就可以了，没有必要去创建对象来调用方法。

  14. final

    用于修饰类、方法、变量

    类：被修饰的类，不能被继承。

    方法：被修饰的方法，不能被重写。

    变量：被修饰的变量，有且仅能被赋值一次。

      变量名=new 类名（）；

    饿汉式单例
    在使用类的时候就已经将对象创建完毕，无论是否用到该实例化对象

    1. 私有构造方法

      }

  17. 接口

    描述所应该具备的方法，具体的实现由接口的实现类来完成。

    使用interface关键字定义接口

    特点：
    接口中的方法均为公共访问的抽象方法，程序默认有public abstract
    接口中无法定义普通成员变量，均为静态常量，程序默认有public static final，需赋初始值

    接口的实现：
    接口不可以创建对象，但可以被类所实现，关键字为implements。
    如果一个类实现了接口，但没有重写接口的所有抽象方法，这个类也必须定义为抽象类
